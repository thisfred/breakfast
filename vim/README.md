# vim-breakfast

## Installation

- [vim-plug](https://github.com/junegunn/vim-plug)
  - Add `Plug 'thisfred/breakfast', {'for': 'python', 'rtp': 'vim'}` to .vimrc
  - Run `:PlugInstall`
